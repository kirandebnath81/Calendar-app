## **How to install and run locally ?**

```
$ git clone https://github.com/kirandebnath81/Calendar-app.git
$ cd Calendar-app
$ npm install
$ npm start
```

## **Features -**

- Get all dates
- Get today date
- Change monthly
- Change yearly
- Manage Events
  - Create
  - Edit
  - Delete
  - View
- Fully Responsive
